#########################################
job2q: Submit any job to any queue
#########################################

job2q is a python package to efortlessly submit jobs to HPC clusters, it currently supports the following job schedulers and simulation packages:

+------------------+------------------+
| Schedulers       | Packages         |
+==================+==================+
| - LSF            | - VASP           |
| - Slurm          | - DFTB+          |
|                  | - ORCA           |
|                  | - Gaussian       |
|                  | - deMon2k        |
+------------------+------------------+
