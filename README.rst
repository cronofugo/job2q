#######################################
JobToQueue: Submit any job to any queue
#######################################

JobToQueue is a python package to readily submit jobs to Linux clusters, it currently supports the following schedulers and simulation packages:

+------------------+------------------+
| Schedulers       | Packages         |
+==================+==================+
|- PBS             | - Gaussian       |
|- LSF             | - VASP           |
|- Slurm           | - DFTB+          |
|                  | - Orca           |
|                  | - deMon2k        |
+------------------+------------------+
