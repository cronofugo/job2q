#######################################
JobToQueue: Submit any job to any queue
#######################################

JobToQueue is a python package to readily submit jobs to Linux clusters.

It currently supports the following schedulers

- PBS
- LSF
- Slurm

and the following simulation packages

- Gaussian
- VASP
- DFTB+
- Orca
- deMon2k
- AutoDock
- NAMD
