#########################################
job2q: Submit any job to any queue
#########################################

job2q is a python package to efortlessly submit jobs to HPC clusters, it currently supports LSF and Slurm schedulers and the following simulation programs:

+------------------+------------------+
| Schedulers       | Packages         |
+==================+==================+
| - LSF            | - VASP           |
| - Slurm          | - DFTB+          |
|                  | - ORCA           |
|                  | - Gaussian       |
|                  | - deMon2k        |
+------------------+------------------+

- VASP
- DFTB+
- ORCA
- Gaussian
- deMon2k
- Autodock
