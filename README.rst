#######################################
Submit any job to any queue
#######################################

JobToQueue is a python package to readily submit jobs to Linux clusters, it currently supports the following job schedulers and simulation packages:

+------------------+------------------+
| Schedulers       | Packages         |
+==================+==================+
| - LSF            | - VASP           |
| - Slurm          | - DFTB+          |
|                  | - ORCA           |
|                  | - Gaussian       |
|                  | - deMon2k        |
+------------------+------------------+
